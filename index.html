// Projects

Projects

MatLLM — LLM Fine-tuning for Materials Science
Fine-tuning large language models on materials science literature to extract process–property relationships and automate data mining from academic papers. Implements a RAG pipeline over domain-specific corpora.
LLMRAG Hugging FacePyTorch
↗ GitHub
MicroDiff — Conditional Diffusion for SEM Images
Conditional DDPM for SEM microstructure image synthesis. Generates realistic microstructures conditioned on LPBF process parameters to augment limited experimental datasets.
DDPMDiffusion Model PyTorchSEM
↗ GitHub
AM Process Optimizer
Bayesian optimization pipeline for LPBF process parameter tuning. Minimizes porosity while maximizing tensile strength, integrating surrogate models with FEM simulations.
Bayesian Opt.LPBF FEMPython
↗ GitHub
MatBench Runner
Benchmarking framework for ML models on materials property prediction tasks. Standardizes data splits, preprocessing pipelines, and evaluation metrics across public datasets.
BenchmarkingGNN Python
